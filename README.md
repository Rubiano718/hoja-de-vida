# Hoja de Vida Web

## Descripción
Este proyecto consiste en una hoja de vida desarrollada como página web.

## Tecnologías utilizadas
- Next.js
- Tailwind CSS

## Funcionalidades
- Visualización de información personal
- Diseño responsive
- Secciones organizadas
